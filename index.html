Red Hat AsciiDoc Templates

This repo is used to create and test the AsciiDoc templates used by Red Hat's documentation platform. See the repo for more information.

Here are some demo pages:

  • AsciiDoc Styleguide - A listing of all elements that can show up in an AsciiDoc.
  • Code Samples - A list of all of the supported code formats and what they look like when syntax highlighting is applied.
  • A Sample 'Assembly' - A sample document that has been modified for testing purposes.
